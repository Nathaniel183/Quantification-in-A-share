A股量化策略仓库

📋 项目简介

本仓库用于存放A股量化策略，逐步完善并持续更新量化策略回测情况以及选股情况。

⚠️ 重要声明

- 所有结果完全由程序自动生成输出
- 数据及结果仅供参考
- 不作为任何投资建议
- 投资有风险，入市需谨慎

📁 仓库结构

主目录

- result/- 策略运行结果目录
- 其他文件 - 策略程序代码文件

策略结果目录结构 (result/策略名/)

    result/
    ├── 策略A/                      # 策略A的运行结果
    │   ├── 说明.txt               # 策略相关详细说明文档
    │   ├── 收益曲线.jpg           # 策略收益曲线可视化图表
    │   └── 详情记录/              # 详细运行结果记录
    │       ├── 202403.csv        # 2024年3月运行结果
    │       ├── 202404.csv        # 2024年4月运行结果
    │       ├── ...
    │       └── 最新时期.csv       # 最新时期的运行结果
    │
    ├── 策略B/
    │   ├── 说明.txt
    │   ├── 收益曲线.jpg
    │   └── 详情记录/
    │       └── ...
    │
    └── ... (其他策略)

📄 文件说明

1. 策略说明文件 (说明.txt)

- 策略的基本原理和逻辑
- 参数设置和调优过程
- 策略的选股方式和条件
- 运行周期和频率说明
- 风险评估和注意事项

2. 收益曲线图 (收益曲线.jpg)

- 策略历史收益表现可视化
- 与基准（如上证指数）对比
- 回撤情况展示

3. 详细记录文件 (详情记录/时期.csv)

- 每个运行时期的详细结果
- 包含最新时期的运行结果

📊 CSV文件内容格式

每个时期CSV文件包含以下列：

  列名     	说明       	示例     
  股票代码   	6位数字代码   	000001 
  因子/指标  	因子/指标计算值 	F-Score
  group_n	因子/指标排序分组	3      
  股票名称   	股票中文名称   	平安银行   
  所属行业   	股票所属行业   	银行     
  仓位     	该股票当期仓位  	10%    

🔄 更新频率

- 策略回测结果：定期更新
- 选股结果：按策略设定周期更新
- 收益曲线：每月更新一次
- 代码优化：持续进行

📈 策略分类

- 因子选股策略
- 趋势跟踪策略
- 均值回归策略
- 多因子组合策略
- 机器学习策略

🚀 使用方法

1. 查看感兴趣的策略目录
2. 阅读说明.txt了解策略详情
3. 查看收益曲线.jpg了解历史表现
4. 查看最新时期的CSV文件获取当前选股结果
5. 可追溯历史记录查看策略稳定性
